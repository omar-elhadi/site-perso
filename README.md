# Omar El Hadi - Portfolio Website 🚀

Personal portfolio showcasing full-stack development expertise.

## ✨ Live Site

[View Live Demo](https://omar-elhadi.dev/) | [GitHub Repository](https://github.com/omar-elhadi/site-perso)

## 💻 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Styling:** Tailwind CSS (via CDN), Custom CSS
- **Icons:** Feather Icons

## 📁 Project Structure
```
site-perso/
├── assets/           # Images & media files
│   ├── Img.png       # Profile photo placeholder
│   └── ...            # Other images (chatty, cinema, etc.)
├── components/        # Reusable HTML/CSS components  
├── index.html         # Main portfolio page with SEO optimization
├── style.css          # Custom styles & animations
└── script.js          # Interactive features & dynamic content

## 🎯 Features
- Dark-themed responsive design
- Typewriter effect for domain expertise showcase  
- Animated project cards with smooth transitions
- Contact form integration (placeholder)

## 🔧 Development Setup (if modifying locally)

```bash
# No build process required - just open index.html in browser!

# To serve locally:
python3 -m http.server 8000 # Or use your preferred static server
```

## 🚀 Deployment via GitHub Pages
This site is automatically deployed to **GitHub Pages** on every push.  
CI/CD handled by [`.github/workflows/deploy.yml`](./.github/workflows/deploy.yml) (assumed).
